new XMLHttpRequest() → creates a new instance of the XMLHttpRequest object

.open(args) → initializes a request
	args: type of request (GET, POST..), url, isAsynchronousRequest

.send(arg) → sends the request
	arg: content to be send on a POST request

.setRequestHeader(args) → sets the value of an HTTP request header, which contains information about the sender and the request.
	args: key-value pair of the information to be sent

.onload → event handler, where you handle the response

.onreadystatechange → event listener that handles achange in the state of the request
	xhr.readyState = 4 - operation is complete
	xhr.status = 201 - successful request
