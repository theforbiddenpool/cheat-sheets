JQUERY
$ → dollar sign operator, invoques the jQuery

$("button") → selects all button elements. You can use CSS selectors.
$(".target:nth-child(n)") → get all the nth elements
$(".target:odd") → target elements based on their positions. Can also use :even. jQuery is zero-index, so the :even will get the 1, 3, etc. element and the :odd will get 2, 4, etc. element.

FUNCTIONS
addClass("name-class..") → adds a new class
removeClass("name-class..") → removes a class
css("prop", "value") → change the CSS of an element
prop("prop", "value") → changes the property of elements
html("...") → lets you add HTML tags and text within an element
text("...") → only alters text without adding tags
remove() → removes an element from the DOM
appendTo("#element") → appends an element to another one
clone() → makes a copy of an element

parent() → get parent of whichever element you've selected
children() → get the children of whichever element you've selected
